**Avalanche スターターキット** には、Avalanche 開発をすぐに始めるために必要なツールがすべて含まれています。  
その中でも特に重要なツールが **Avalanche CLI** です。  

このツールを使用すると、以下のようなことが可能になります。  

- **ローカルネットワークの構築**  
- **Avalanche L1（レイヤー1ブロックチェーン）の作成**  
- **Avalanche L1/VM の設定カスタマイズ**  

このコースでは、 **GitHub 上のホスト環境で Avalanche スターターキットを実行** します。  
これが最も簡単かつ迅速に開発を開始する方法です。  

<Steps>  
<Step>  

### Avalanche Starter Kit の GitHub リポジトリを開く  

以下のリンクからリポジトリにアクセスしてください。  

<Link  
    href="https://github.com/ava-labs/avalanche-starter-kit"  
    className={cn(  
    buttonVariants({ variant: 'default', className: 'mt-4' }),  
    )}  
    target='_blank'  
>Open Avalanche Starter Kit</Link>  

</Step>  
<Step>  

### Codespace を作成する  

GitHub 上で Avalanche Starter Kit の開発環境（Codespace）を作成するには、以下の手順を実行してください。  

1. **緑色の「Code」ボタン** をクリック  
2. **「Codespaces」タブ** に切り替える  
3. **「Create Codespace on main」** をクリック  
   - すでに Codespace を持っている場合は、 **「+（プラス）」ボタン** をクリック  

作成が完了すると、新しいタブで Codespace が開きます。  
ビルドが完了するまで数分待ちましょう。  

</Step>  
<Step>  

### 全てが正しく動作しているか確認する  

ターミナルを開くには、以下の方法のいずれかを使用してください。  

- **キーボードショートカット** ： `Ctrl + \` `  
- **メニューから開く**  

![](/common-images/avalanche-starter-kit/new-terminal.png)  

次に、以下のコマンドを入力し、Avalanche CLI が正しく動作するか確認します。  

```bash  
avalanche -h  
```  

このコマンドを実行すると、以下のような **Avalanche CLI のヘルプメッセージ** が表示されるはずです。  

```
Avalanche-CLI is a command-line tool that gives developers access to
everything Avalanche. This release specializes in helping developers
build and test Avalanche L1s.

To get started, look at the documentation for the subcommands or jump right
in with avalanche blockchain create myNewAvalanche L1.

Usage:
  avalanche [command]

Available Commands:
  config                Modify configuration for Avalanche-CLI
  help                  Help about any command
  key                   Create and manage testnet signing keys
```  

</Step>  
<Step>  

### （オプション）Codespace を Visual Studio Code で開く  

ブラウザ上の IDE ではなく、 **Visual Studio Code（VS Code）** で Codespace を開くことも可能です。  

![](/course-images/codespaces/open-locally.png)  

最初に切り替える際、次の操作が必要になる場合があります。  

- **[Codespaces 拡張機能](https://marketplace.visualstudio.com/items?itemName=GitHub.codespaces)** をインストール  
- **GitHub アカウントと VS Code を接続**（未接続の場合）  

これにより、ローカル環境でより柔軟な開発が可能になります。  

</Step>  
</Steps>
