import { GenerateKeysButton, SignMessageButton, VerifySignatureButton } from '@/components/signatureSchemes'
import { Steps, Step } from 'fumadocs-ui/components/steps'

署名スキームを実際に使ってみましょう。以下のステップを実行します。  

- **鍵ペアの生成**: 公開鍵と秘密鍵のペアを作成します。  
- **メッセージの署名**: 秘密鍵を使ってメッセージに署名し、署名データを生成します。  
- **署名の検証**: 公開鍵を使って、メッセージの署名を検証します。  

<Steps>  
<Step>  

### 鍵を生成する  
まずは、公開鍵と秘密鍵のペアを生成します。  

<GenerateKeysButton />  

</Step>  
<Step>  

### メッセージに署名する  
生成した秘密鍵を使って、メッセージに署名を行います。好きなメッセージを入力してください（例: "Hello"）。  

<SignMessageButton />  

</Step>  
<Step>  

### 署名を検証する  
公開鍵があれば、誰でも署名の正当性を確認できます。これにより、メッセージの **真正性、否認防止、完全性** が保証されます。  

<VerifySignatureButton />  

</Step>  
</Steps>
