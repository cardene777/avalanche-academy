ラウンドトリップ通信（往復メッセージ）を可能にする **受信コントラクト** をデプロイします。  

<Steps>  

<Step>  

### **受信コントラクトをデプロイする**  

以下のコマンドを実行して、修正した受信コントラクトを Avalanche L1 にデプロイします。  

```bash
forge create --rpc-url myblockchain --private-key $PK contracts/interchain-messaging/send-roundtrip/receiverOnSubnet.sol:ReceiverOnSubnet
```

実行すると、以下のような出力が得られます。  

```
[⠊] Compiling...
[⠢] Compiling 1 files with Solc 0.8.18
[⠆] Solc 0.8.18 finished in 101.71ms
Compiler run successful!
Deployer: 0x8db97C7cEcE249c2b98bDC0226Cc4C2A57BF52FC
Deployed to: 0x17aB05351fC94a1a67Bf3f56DdbB941aE6c63E25 // [!code highlight]
Transaction hash: 0x11df9e14bdae4af60a618a900faa955d75e0ce7dd0f2ccc1ea6a764c149ef805
```

</Step>  

<Step>  

### **受信コントラクトのアドレスを更新する**  

デプロイ後、新しい受信コントラクトのアドレスを環境変数に保存します。  

```bash
export RECEIVER_ADDRESS=0x17aB05351fC94a1a67Bf3f56DdbB941aE6c63E25
```

</Step>  

</Steps>
