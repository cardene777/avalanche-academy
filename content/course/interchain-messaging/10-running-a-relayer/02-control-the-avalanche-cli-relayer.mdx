Avalanche Starter-Kit を使用して Avalanche CLI で独自の L1 チェーンをデプロイすると、**起動時にリレイヤーを実行するオプションが提供されます。** これにより、ローカル環境で動作する Avalanche L1 間のメッセージを自動的に送信するリレイヤーが起動されます。  

設定方法については次のセクションで詳しく説明しますが、ここでは基本的な操作方法を学びます。  

## リレイヤーの停止  

リレイヤーを停止したい場合は、以下のコマンドを実行します。  

```bash
avalanche teleporter relayer stop
```

## リレイヤーの起動  

リレイヤーを再び起動する場合は、以下のコマンドを実行します。  

```bash
avalanche teleporter relayer start
```
