Avalanche Warp Messaging (AWM) リレイヤーのバイナリは、**複数の方法で取得可能** です。  

- **GitHub リリースページからダウンロード:**  

各リリース時に、バイナリが [GitHub のリリースページ](https://github.com/ava-labs/awm-relayer/releases) にアップロードされます。  

- **Docker を使用:**  

[Docker Hub](https://hub.docker.com/r/avaplatform/awm-relayer) に `avaplatform/awm-relayer:latest` のタグでイメージが公開されています。  

- **ソースコードからビルド:**  

直接ソースコードをビルドすることも可能です。これにより、**独自の変更を加えたり、異なるプラットフォームで実行することができます。** ソースコードは [GitHub リポジトリ](https://github.com/ava-labs/awm-relayer?tab=readme-ov-file#building) で公開されています。  

## Avalanche Starter Kit を利用する場合  

[Avalanche Starter Kit](https://github.com/ava-labs/avalanche-starter-kit) を使用する場合、リレイヤーは **Dev Container** にすでに含まれています。リレイヤーを起動するには、以下のコマンドを実行してください。  

```bash
avalanche teleporter relayer start
```
