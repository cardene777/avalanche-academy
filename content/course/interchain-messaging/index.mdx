このコースでは、Interchain Messaging と Avalanche Warp Messaging を使用して、クロスチェーンL1（Layer 1）対応の Solidity dApp を構築する方法を学びます。

## なぜこのコースを受講するのか？

ブロックチェーン技術の大きな進歩の一つに、Avalanche のような **マルチチェーンシステム** の開発があります。これにより、スケーラビリティ（拡張性）、相互運用性、柔軟性が大幅に向上しました。マルチチェーンシステムの核となるのは、複数のブロックチェーンが相互に通信できることです。それぞれのチェーンの仮想マシン（VM）は特定のユースケースに最適化されており、ネットワーク全体のパフォーマンスを向上させます。

**クロスチェーン通信** は、マルチチェーンシステムにおける重要な要素です。Interchain Messaging と Avalanche Warp Messaging を利用すれば、既に監査された開発フレームワーク上に構築できるため、クロスチェーンL1 dApp を容易に開発できます。

### **相互運用性（Interoperability）**

最初のセクションでは、マルチチェーンシステムにおける **相互運用性（interoperability）** の基本概念を学びます。異なるブロックチェーン間での相互運用の例や、「送信元（source）」、「宛先（destination）」、「メッセージ（message）」といった基本的な用語について説明します。

### **Avalanche Interchain Messaging**

このセクションでは、Avalanche Interchain Messaging の仕組みや、一般的な dApp 開発者が意識する必要のない部分について学びます。また、実際に最初の **クロスチェーンL1 dApp** を構築します。

### **クロスチェーン通信のセキュリティ**

このセクションでは、クロスチェーン通信を安全に行うための技術について学びます。署名スキーム（signature schemes）、マルチシグ（multi-signature schemes）、BLSマルチシグ（BLS multi-signature scheme）について詳しく掘り下げます。

### **Avalanche Interchain Messaging プロトコル**

Avalanche のブロックチェーン同士は、**Avalanche Warp Messaging（AWM）** を利用してネイティブに相互運用できます。このセクションでは、AWM の **メッセージフォーマット** や **メッセージの流れ（message flow）** について学びます。

## **受講前の前提知識**

### **Avalanche の基礎知識**

このコースは、Avalanche の基本概念を十分に理解している方向けです。以下の知識が必要になります。

- **仮想マシン（Virtual Machines）：** 仮想マシンとは何か、カスタマイズの意味
- **Avalanche L1 とブロックチェーンの違い：** **VM（仮想マシン）**、**Blockchain（ブロックチェーン）**、**Avalanche L1** の違い

これらの理解が不足している場合は、**Avalanche Fundamentals（Avalancheの基礎）** や **Multi-Chain Architecture（マルチチェーンアーキテクチャ）** のコースを先に受講することを推奨します。

### **ソフトウェア開発の基礎知識**

このコースでは多くのコードを書く必要はありませんが、ある程度の理解が求められます。以下の技術について知識があるとスムーズに学習できます。

- **Solidity:** 言語の基本概念に精通していること。演習では **クロスチェーンサブネット（cross-subnet）** dApp を Solidity で記述します。
- **Docker:** コース後半の高度な演習では、セットアップを簡素化するため **コンテナ環境** を利用します。Docker や Docker Compose の基本概念に慣れていると理解しやすくなります。
- **テスト（Testing）:** 単体テスト（unit testing）の経験や、テスト手法に関する基本的な知識があると理想的です。

## **学習目標**

このコースを修了すると、以下の内容を習得できます。

- **クロスチェーン通信の課題を理解する**
- **Avalanche Warp Messaging が他のクロスチェーン通信プロトコルとどう違うのかを知る**
- **Avalanche Warp Messaging と Teleporter の違いを理解する**
- **クロスAvalanche L1 dApp（例えば資産ブリッジ）を構築するスキルを身につける**

全体を通して、本コースは **Teleporter の高度な理解** を提供することを目的としています。修了することで、**Avalanche L1 を跨ぐブロックチェーンアプリケーションの開発に備えることができます**。
